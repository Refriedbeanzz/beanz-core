# Beanz Core

Beanz Core is the foundational framework mod for the RefriedBeanz Hytale server.

This mod provides the base systems that other mods will depend on, including:

- configuration management
- player data storage
- region detection
- logging and debugging tools

The goal of Beanz Core is to act as the central framework for building advanced server systems such as:

- skill progression
- faction territory
- raid events
- creature taming
- world progression

Future modules will include:

- beanz-skills
- beanz-raids
- beanz-factions
- beanz-taming
- beanz-world
